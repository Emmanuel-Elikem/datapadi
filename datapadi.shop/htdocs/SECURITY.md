# Security Policy

## Reporting Security Issues

If you discover a security vulnerability, please email: support@datapadi.shop

## About DataPadi

DataPadi is a legitimate data bundle service for mobile networks in Ghana.

- **Service**: Mobile data bundle sales
- **Networks Supported**: MTN, AirtelTigo, Telecel
- **Country**: Ghana
- **Payment**: Paystack (secure payment gateway)

## No Malicious Content

This website:
- Does NOT contain malware
- Does NOT phish for credentials
- Does NOT engage in social engineering
- Only processes legitimate data bundle purchases

## Admin Section

The `/admin/` directory is a legitimate administrative interface protected by authentication. It is not accessible to public users and does not contain any malicious content.

## Contact

For any security concerns, contact us at support@datapadi.shop
